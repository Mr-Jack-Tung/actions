usage: sema4ai.actions [-h] {run,list} ...

Sema4.ai framework for AI actions.

positional arguments:
  {run,list}
    run       Collects actions with the @action decorator and all actions that
              matches based on the action name filter.
    list      Provides output to stdout with json contents of the actions
              available.

options:
  -h, --help  show this help message and exit

View https://github.com/sema4ai/actions for more information
